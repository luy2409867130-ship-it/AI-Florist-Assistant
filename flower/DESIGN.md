# AI花店助手 - 登录页面设计说明

## 一、项目概述

AI花店助手是一个为花店提供智能化辅助的工具平台，旨在通过人工智能技术提升花店运营效率和客户服务质量。本设计文档针对登录页面进行详细说明。

---

## 二、设计目标

1. **品牌展示**：突出花店主题，传递温馨、优雅的品牌形象
2. **用户体验**：提供简洁、流畅的登录流程
3. **响应式设计**：适配多种设备尺寸
4. **安全性**：提供基本的表单验证和密码保护

## 五、交互设计

### 5.1 表单验证

| 验证项 | 规则 | 错误提示 |
| :--- | :--- | :--- |
| 用户名/手机号 | 必填 | 请输入用户名或手机号 |
| 用户名/手机号 | 格式校验 | 请输入有效的用户名或手机号 |
| 密码 | 必填 | 请输入密码 |
| 密码 | 最小长度6位 | 密码长度不能少于6位 |
| 密码 | 最大长度20位 | 密码长度不能超过20位 |

### 5.2 交互效果

1. **输入框聚焦**：边框颜色变为主题色，显示阴影效果
2. **输入框失焦**：根据验证结果显示成功/失败状态
3. **密码显示/隐藏**：点击眼睛图标切换密码可见性
4. **记住我**：复选框选中状态可保存到本地存储
5. **登录按钮**：点击后显示加载状态，禁止重复点击
6. **链接点击**：显示功能开发中的通知提示
7. **背景动画**：花朵装饰元素浮动动画效果

### 5.3 本地存储

| 存储项 | 说明 |
| :--- | :--- |
| ai-flower-username | 保存用户名/手机号 |
| ai-flower-remember | 记住我状态 |

## 六、响应式适配

### 6.1 断点设置

| 屏幕宽度 | 适配策略 |
| :--- | :--- |
| > 480px | 标准桌面端布局 |
| ≤ 480px | 移动端布局，减少内边距 |
| ≤ 600px（高度） | 紧凑型布局，适应小屏设备 |

### 6.2 移动端优化

- 卡片内边距：40px → 30px
- 品牌标题字号：28px → 24px
- 图标尺寸：80px → 65px
- 社交登录按钮：横向排列 → 纵向排列

## 八、功能扩展预留

登录页面已为后续功能扩展预留以下接口：

1. **第三方登录**：预留微信、支付宝等登录入口
2. **验证码登录**：可扩展手机验证码登录方式
3. **多语言支持**：页面结构支持国际化扩展
4. **路由跳转**：登录成功后跳转到dashboard页面
