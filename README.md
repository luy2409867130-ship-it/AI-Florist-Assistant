# AI-Florist-Assistant
An AI-powered florist assistant built with DeepSeek API.
主要提供：
- 花束推荐
- AI 文案生成
- Dashboard
- AI 历史记录

---

## 技术栈

- HTML
- CSS
- JavaScript
- DeepSeek API
- Git
